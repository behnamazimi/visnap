ViSnap supports Docker for consistent testing environments and integrates with popular CI/CD platforms.

## Docker Support

Run ViSnap tests in Docker for consistent results across different environments.

### Docker Usage Model

Use Docker via CLI flags. You typically do not configure Docker in `visnap.config.ts`.

```npm
# Run tests inside Docker
npx visnap test --docker
```

```npm
# Update baselines inside Docker
npx visnap update --docker
```
