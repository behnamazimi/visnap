---
title: Viewports
description: Single and multi-viewport setup
---

Test across different screen sizes to ensure your UI works on various devices.

## Single Viewport

Test with one viewport size:

```typescript
viewport: {
  desktop: { width: 1920, height: 1080 }
}
```

## Multiple Viewports

Test across multiple screen sizes:

```typescript
viewport: {
  desktop: { width: 1920, height: 1080 },
  tablet: { width: 768, height: 1024 },
  mobile: { width: 375, height: 667 }
}
```

## Per-Test Viewports

Override viewports for specific test cases using adapter-specific configuration:

- **Storybook**: Use `parameters.visualTesting.viewport`
- **URL Adapter**: Use `viewport` property in URL configuration

See the specific adapter documentation for details.
